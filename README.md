## Building
Hello Mr Mozilla addon manual reviewer I'm using node `v16.16.0` and pnpm `6.11.0`.

#### Clone the repository
`git clone https://github.com/ma1ted/githubmoji`

#### `cd` into the new directory
`cd githubmoji`

#### Install the required packages
`pnpm install`

#### Bundle the code
`pnpm run build`
